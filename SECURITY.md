# Security Policy

## Supported versions

Only the latest published version is supported.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report via [GitHub private advisory](https://github.com/adaofeliz/sure-mcp/security/advisories/new) or email the maintainer directly.

## Scope

In scope:
- Vulnerabilities that could expose the `SURE_API_KEY` to unintended parties
- Vulnerabilities that could expose Sure financial data
- Injection attacks via tool input parameters

Out of scope:
- Issues that require the attacker to already have access to the user's machine or MCP client config
- Issues in the Sure API itself (report those to the Sure project)
- Social engineering

## Response

Acknowledged within 72 hours. Fix timeline depends on severity.
